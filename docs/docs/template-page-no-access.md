# Notion 模板页面不存在或无访问权限

如果插件提示 `Notion 模板页面不存在或无访问权限`，通常不是插件故障，而是模板页面已经失效，或者 Notion 集成对该页面的权限丢失了。

## 解决方法

打开 Notionhub 插件，进入同步服务配置页，点击 `打开 Notion 模板`，确认 Notion 是否能正常打开该模板页面。

![](https://images.malinkang.com/2026/03/b76dc591d8373d9aed0e80d301090835.png)

如果模板已经被删除，你可以：

- 重新复制一份新的模板
- 恢复已经删除的模板

如果模板没有被删除，可以点击 `重新复制模板`，然后选择之前的模板。

如果模板没有被删除，你可以重新执行一次模板授权流程：

- 打开 Notionhub 插件
- 点击 `复制 Notion 模板`
- 按提示选择你之前使用的那个模板
- 打开该模板页面，点击右上角三个点，选择 `拷贝链接`
- 回到插件，粘贴模板链接并提交

## 仍然无法解决？

如果重新复制模板、重新授权之后仍然提示这个错误，请联系[技术支持](./support.md)处理，并把当前使用的模板链接发给我们。
