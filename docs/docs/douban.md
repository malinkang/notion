# 使用指南

使用之前没有安装的请先参考[安装指南](./install.md)安装Notionhub插件

预览效果：https://movie.malinkang.com/



### 1. 点击打开插件，然后点击去使用

![](https://images.malinkang.com/2025/12/07e6517c68f3c5dd0a3835164af64a79.png)



### 2.输入激活码，点击激活。

![](https://images.malinkang.com/2025/12/642d0c1e05c921309187a87eaf12b5af.png)

### 3.复制模板链接

点击复制Notion模板按钮，会打开一个链接

![](https://images.malinkang.com/2025/08/b8050c4503c8f0bd740b221e1e7fccc7.png)

点击下一步

![](https://images.malinkang.com/2025/12/b09354594b9fdceed48608335c619211.png)

选择使用`开发人员提供的模板`，然后点击`允许访问`访问即可。

![](https://images.malinkang.com/2025/12/1a99fb1f0ba7e53a1829a4029e58091b.png)

### 4. 输入豆瓣用户名

打开豆瓣：https://www.douban.com/， 然后点击个人主页，查看浏览器里的网址，people/后面到下一个/之间的字符串就是用户名。比如我的是 https://www.douban.com/people/malinkang/?_i=4571642luvf7GJ， 那么我的用户名就是malinkang。获取成功之后回到插件输入并点击提交按钮。

![](https://images.malinkang.com/2025/12/8c0844e554a51814562a26f4e26720fa.png)

![](https://images.malinkang.com/2025/12/cd9ec31dca3d7d3168564c8e33daa5ff.png)

![](https://images.malinkang.com/2025/12/1c6ab2fe6340c2fd07c766c22ced2723.png)

### 5. 运行

扫码登录完成会跳转到运行页面，点击`开始同步豆瓣`按钮开始同步。

![](https://images.malinkang.com/2025/12/c54d7a9706231fd9e22c2111222abdfd.png)

### 6.查看数据

点击运行之后，可以点击右上角的锯齿图标，然后点击打开Notion模板，可以查看模板里是否有数据。一般一两分钟左右就会有数据。

![](https://images.malinkang.com/2025/12/6bb4dac8b76363378c3957e272d06099.png)
