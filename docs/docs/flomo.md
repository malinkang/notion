# flomo使用指南

## 功能介绍

Notionhub目前已经支持将Flomo同步到Notion了，模板样式参考了flomo网页版的设计，支持如下功能：

1. 超链接和标签样式保持和官网一致
2. 图片和音频文件支持上传到Notion。（免费Notion用户会有5M限制，超过5M会上传失败）
3. 多图支持一行3图展示
4. 支持关联笔记
5. 支持多级标签
6. 支持热力图和按照年月日周笔记数的汇总。

## 使用

### 1. 点击打开插件，然后点击去使用

![](https://images.malinkang.com/2026/01/93952fd086e02ca9592cb79cbc15d01a.png)



### 2.输入激活码，点击激活。

![](https://images.malinkang.com/2026/01/e127413136534132477edbcfd78f50f7.png)

### 3.复制模板链接

点击复制Notion模板按钮，会打开一个链接

![](https://images.malinkang.com/2025/08/b8050c4503c8f0bd740b221e1e7fccc7.png)

点击下一步

![CleanShot 2026-01-15 at 11.02.23@2x](https://images.malinkang.com/2026/01/5a97b74dc8e870920cfc881ece646002.png)

选择使用`开发人员提供的模板`，然后点击`允许访问`访问即可。

![](https://images.malinkang.com/2026/01/1db3a87b03d89e2d9d0c5ba17f0ad22e.png)



### 4. flomo登录

授权完成，会跳转到flomo登录页面，点击登录flomo，跳转到flomo网页版本，登录flomo网页版即可，然后回到插件。如果网页版之前已经登录了，直接返回插件即可。再次回到插件会展示你的flomo信息，然后点下一步即可。

![](https://images.malinkang.com/2026/01/165846b95b6b218fddc6d6966c4f1112.png)

![](https://images.malinkang.com/2026/01/d76560f3b31763dc04b67705d83e1ee1.png)

### 5. 运行

扫码登录完成会跳转到运行页面，点击`开始同步flomo`按钮开始同步。同步在服务器上进行，关闭电脑也不影响同步。目前是每天6/12/18/24，4个时间段

![](https://images.malinkang.com/2026/01/ee0f2dc7d8bb5a04e7f99a273bfb9fdf.png)

### 6.查看数据

点击运行之后，可以点击右上角的锯齿图标，然后点击打开Notion模板，可以查看模板里是否有数据。一般一两分钟左右就会有数据。

![](https://images.malinkang.com/2026/01/2d570e45993ed59c8e176e6969aca106.png)
