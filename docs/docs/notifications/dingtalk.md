# 钉钉通知配置

NotionHub 可以通过钉钉群机器人发送同步通知。钉钉通知通常使用群机器人的 Webhook 地址。

配置钉钉通知前，需要先准备：

- Webhook 地址
- 安全设置。如果开启了加签，还需要 Secret

## 一、如何申请

### 1. 创建钉钉群机器人

在钉钉中打开用于接收通知的群聊。

进入群设置，找到 `机器人` 或 `智能群助手`，添加 `自定义机器人`。

填写机器人名称，例如：

```text
NotionHub 通知
```

### 2. 配置安全设置

钉钉自定义机器人一般需要配置安全策略，常见方式包括：

- 关键词
- 加签
- IP 地址

建议先使用 `关键词`，例如填写：

```text
NotionHub
```

这样通知内容里包含 `NotionHub` 时，机器人就可以发送消息。

如果你选择 `加签`，创建机器人后需要同时保存：

- Webhook 地址
- Secret

### 3. 复制 Webhook

机器人创建完成后，钉钉会生成一个 Webhook 地址，格式大致如下：

```text
https://oapi.dingtalk.com/robot/send?access_token=xxxx
```

请保存这个地址，后续需要填写到 NotionHub 插件中。

## 二、如何在插件中配置

1. 打开 NotionHub 浏览器插件。
2. 进入需要接收通知的同步服务。
3. 找到通知配置区域。
4. 通知平台选择 `钉钉`。
5. 填写钉钉机器人 Webhook 地址。
6. 如果插件提供 Secret 输入框，并且你在钉钉中开启了加签，请填写 Secret。
7. 点击保存或提交。
8. 运行一次同步，检查钉钉群是否收到通知。

## 常见问题

如果收不到通知，优先检查：

- Webhook 地址是否复制完整。
- 钉钉安全设置是否和插件配置一致。
- 如果使用关键词，通知内容是否包含对应关键词。
- 如果使用加签，Secret 是否填写正确。
