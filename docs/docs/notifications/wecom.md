# 企业微信通知配置

NotionHub 可以通过企业微信群机器人发送同步通知。企业微信通知通常使用群机器人的 Webhook 地址。

配置企业微信通知前，需要先准备：

- Webhook 地址

## 一、如何申请

### 1. 创建企业微信群机器人

在企业微信中打开用于接收通知的群聊。

进入群设置，找到 `群机器人`，添加机器人。

填写机器人名称，例如：

```text
NotionHub 通知
```

### 2. 复制 Webhook

机器人创建完成后，企业微信会生成一个 Webhook 地址，格式大致如下：

```text
https://qyapi.weixin.qq.com/cgi-bin/webhook/send?key=xxxx
```

请保存这个地址，后续需要填写到 NotionHub 插件中。

如果后续找不到 Webhook，可以回到群机器人管理页面，查看机器人详情并复制。

## 二、如何在插件中配置

1. 打开 NotionHub 浏览器插件。
2. 进入需要接收通知的同步服务。
3. 找到通知配置区域。
4. 通知平台选择 `企业微信`。
5. 填写企业微信机器人 Webhook 地址。
6. 点击保存或提交。
7. 运行一次同步，检查企业微信群是否收到通知。

## 常见问题

如果收不到通知，优先检查：

- Webhook 地址是否复制完整。
- 机器人是否仍在目标群聊中。
- 群聊是否限制了机器人消息。
- 是否复制了正确群聊里的机器人 Webhook。
