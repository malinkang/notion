# 隐私权政策 (Privacy Policy)

**生效日期**: 2026年1月22日

本隐私权政策说明了 **NotionHub**（以下简称“本扩展程序”）如何收集、使用和保护您的信息。

## 1. 我们收集的信息

为了提供同步服务，本扩展程序需要访问以下信息：

*   **Cookie 信息**: 我们需要读取您在 `xiaohongshu.com` 等目标网站的登录状态 (Cookie)，以便代表您获取您自己的笔记、收藏或点赞数据。
*   **用户 ID (UID)**: 用于标识需要同步的特定账号。
*   **Notion 授权信息**: 如果您配置了 Notion 集成，我们会存储您的 Notion API Key 和 Database ID，以便将数据写入您的 Notion 笔记中。

## 2. 数据的存储与使用

*   **本地存储**: 所有敏感信息（如 Token、Cookie、API Key）仅保存在您浏览器的 **本地存储 (Chrome Local Storage)** 中。
*   **数据传输**: 本扩展程序仅在您的浏览器与目标网站（如小红书、Notion）之间直接传输数据。**我们不会将您的任何数据发送到任何第三方服务器，也不会在我们的服务器上收集或存储您的任何个人信息。**
*   **用途限制**: 收集的数据仅用于实现“数据同步”这一核心功能。

## 3. 第三方服务

本扩展程序可能会与以下第三方服务进行交互（仅在您主动发起同步请求时）：

*   **Notion API**: 用于创建和更新笔记页面。
*   **目标网站 API** (如小红书): 用于获取您的笔记内容。

请注意，这些第三方服务受其各自的隐私权政策约束。

## 4. 政策更新

我们会不时更新本隐私权政策。任何更改都将发布在此页面上。

## 5. 联系我们

如果您对本隐私权政策有任何疑问，请通过 GitHub Issues 联系我们。
